Intoduction au language html:

1. Introduction:

le language HTML(hyper text markup language) est un language de marquage , ou encore un language de balisage qui a ete créé en 1991 pour accompagner le web.

dans cette introduction nous alons presenter les balises de base ainsi que les bonnes pratiques HTML.
2.historique du web :
L'historique du language html remonte a l'année 1969:
GML :
premier language de balisage cree en 1969 dans les reseau locaux d'ibm par ses 3 concepteur : goldfarb,mosher et lorie.
SGML :
il s'agit d'une standarisation du language GML,en 1974,pour désigner standard generalized markup language.
WWW :
signifiant Word wide web ayant ete initié en 1989 par tim Berners-lee:
HTML
hypertext markup,issu du SGML et presente la premiere fois par tim berners-lee en 1991, pour etre prose ensuite comme standard de presentation des doc sur le web. differents styles d'affichage de contenu. l'une des principales balises etant la mise en relation entre differentes pages HTML constituant le site ou l'aplication web, a l'aide de la balise d'ancrage (anchor).
